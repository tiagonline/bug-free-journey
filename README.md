# Desafio Técnico - QA Lead (Sigga)
**Autor:** Tiago Oliveira Silva

## Introdução

[cite_start]Este repositório contém a resolução completa do "Desafio Técnico - QA Lead" [cite: 71-73].

Para facilitar a avaliação, o projeto foi organizado em duas áreas principais:

1. **/automacao**: Contém o código-fonte funcional dos projetos de automação (Cypress e K6) [cite: 82-86].
2. **/entregaveis**: Contém toda a documentação estratégica e a análise de métricas [cite: 75-81, 87-99], separada por entregável.

## 1. Plano Estratégico de Testes (Alto Nível)

***Conteúdo:** Contém a análise de arquitetura, pontos de falha, tipos de teste, riscos e os 5 cenários críticos [cite: 75-81].
* **Arquivo:** [**`Ver Entregável 1`**](./entregaveis/entregavel_1.md)

## 2. Automação de Cenários Essenciais

***Conteúdo:** Contém a justificativa técnica para a escolha dos *frameworks* (Cypress e K6) [cite: 82-86]. O código-fonte está na pasta `/automacao`.
* **Arquivo:** [**`Ver Entregável 2`**](./entregaveis/entregavel_2.md)

## 3. Métricas e Análise de Qualidade

***Conteúdo:** Contém a proposta de indicadores (KPIs), o relatório simulado e a análise de métricas acionáveis [cite: 87-92].
* **Arquivo:** [**`Ver Entregável 3`**](./entregaveis/entregavel_3.md)

## 4. Desafios Adicionais (Escala e Privacidade)

***Conteúdo:** Contém a estratégia para testes geolocalizados, paralelos e validação de privacidade (LGPD/GDPR) [cite: 93-99].
* **Arquivo:** [**`Ver Entregável 4`**](./entregaveis/entregavel_4.md)
